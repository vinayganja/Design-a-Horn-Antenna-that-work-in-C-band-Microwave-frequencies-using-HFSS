# Design-a-Horn-Antenna-that-work-in-C-band-Microwave-frequencies-using-HFSS

### Overview
This project involves the **design, modeling, and simulation** of a **C-band pyramidal Horn Antenna** operating in the 4–8 GHz frequency range using **ANSYS HFSS**. The antenna is optimized for microwave applications such as satellite communication, radar systems, and point-to-point microwave links.

### Key Highlights
- **Operating Frequency**: Resonates at **6.01 GHz** (center of C-band)
- **Return Loss**: Excellent impedance matching with **-25.93 dB** at resonant frequency
- **Design Parameters**:
  - Waveguide dimensions: a = 0.795 in, b = 1.0 in, c = 1.59 in
  - Aperture dimensions: Ap = 7.95 in, Bp = 3.975 in
- **Tools Used**: ANSYS HFSS (Finite Element Method)
- **Analysis Performed**:
  - S11 (Return Loss)
  - VSWR
  - Gain & Directivity
  - 3D Radiation Pattern

### Objectives
- Design and simulate a compact horn antenna for C-band applications
- Achieve good impedance matching and radiation characteristics
- Analyze key antenna parameters (gain, directivity, bandwidth, beamwidth)

### Results
The optimized design demonstrates strong performance with low return loss, good directivity, and a focused radiation pattern, making it suitable for practical microwave communication systems.

**Technologies**: HFSS, Electromagnetic Simulation, Antenna Theory
